# Was ist Informatik

Der Begriff Informatik ist im Jahre 1960 entstanden

Informatik ist die Wissenschaft der systematischen Darstellung, Speicherung, Verarbeitung & Übertragung von Informationen. Sie unterteilt sich in die folgenden 4 Fachbereiche:

  * <span style="background-color:#2c2c54">Theoretische Informatik</span>
      * mathematisch logische Grundlagen | Ist ein Problem überhaupt durch Berechnungen lösbar

  * <span style="background-color:#2c2c54">Technische Informatik</span>
      * elektrisch – technische Eigenschaften der Bauteile (Hardware)
        * wichtige Teilgebiete der technischen Informatik ist Schaltalgebra, die Umsetzung logischer Operatoren durch elektronische Schaltungen

  * <span style="background-color:#2c2c54">Praktische Informatik</span>

  * <span style="background-color:#2c2c54">Angewandte Informatik </span>
